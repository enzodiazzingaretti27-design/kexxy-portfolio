# KEXXY Portfolio

Portfolio visual de Enzo Diaz Zingaretti (Kexxy), construido con React, Vite y Tailwind CSS.

## Setup

1. Install Node.js (LTS recommended).
2. Install dependencies:
   - `npm install`
3. Run frontend:
   - `npm run dev`

## Stack

- React + Vite
- Tailwind CSS

## Contenido del sitio

- El contenido principal vive en `src/siteContent.js`.
- Para priorizar la publicación visual, el backend anterior se archivó en `archive/backend-legacy`.
